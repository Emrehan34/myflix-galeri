# MyFlix Gallery

Netflix-stil fotoğraf ve video galerisi uygulaması.

## Özellikler
- Netflix-benzeri koyu tema
- Albüm oluşturma (anında)
- Foto/video yükleme (sürükle-bırak)
- Modal medya görüntüleme
- Responsive mobil uyum
- Giriş/kayıt (e-posta, Google/Apple mock, misafir)

## Kurulum
1. Bu dosyaları GitHub Pages’e veya Netlify’a yükle
2. Tarayıcıdan aç

## Geliştirme
- HTML5 + CSS3 + Vanilla JavaScript
- IndexedDB (medya) + localStorage (meta)
- Font Awesome + Bootstrap Icons
- Inter/Poppins/Montserrat fontlar
