# GitHub Pages ile Yayın Adımları

1) GitHub’a giriş yap
2) Yeni repo oluştur: myflix-gallery (Public)
3) “Add file” → “Upload files”
4) Bu dosyaları seç ve yükle:
   - index.html
   - styles.css
   - script.js
   - README.md
   - .gitignore
   - netlify.toml
5) Commit changes → “Initial commit”
6) Repo ayarları → Pages → Source: Deploy from a branch → Branch: main → Save
7) Birkaç dakika sonra siten şu adreste açılır:
   https://[kullanıcı-adın].github.io/myflix-gallery

Not: Sadece siteye erişim sağlar. Albümlerin farklı cihazlardan senkron olması için Firebase entegrasyonu gerekiyor.
